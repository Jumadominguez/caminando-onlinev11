Este archivo registra todos los cambios realizados localmente antes de hacer commits a GitHub.
Cada entrada debe incluir:
- Fecha y hora
- Tipo de cambio (FEAT, FIX, REFACTOR, etc.)
- Descripción breve del cambio
- Archivos afectados (opcional)

Cuando se solicite subir a GitHub, se usará esta información para crear el mensaje del commit.
